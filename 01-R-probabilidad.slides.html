Probabilidad como frecuencia¶

En su forma ms simple, el concepto de probabilidad puede interpretarse como la frecuencia con que ocurre un evento. Ejemplo: Tirada de dos dados:

Prob(= 2) = 1 / 36 
Prob(= 3) = 2 / 36  
Prob(= 4) = 3 / 36
Prob(= 5) = 4 / 36 
Prob(= 6) = 5 / 36

Prob(= 9) = 4 / 36 
Prob(=10) = 3 / 36
Prob(=11) = 2 / 36

Ejercicio.-- Los soldados mediavales apostaban con dos dados de la siguiente manera: si el resultado es par {2, 4, 6, 8, 10, 12} ganaba el soldado A; y si el resultado es impar {3, 5, 7, 9, 11} ganaba el soldado B. Quin tiene mayor probabilidad de ganar?

Definiciones básicas¶

  • Todas las probabilidades deben estar entre 0 y 1: $$0 <= \text{Prob}(x_i) <= 1$$

  • Las probabilidades de eventos mutuamente exclusivos y colectivamente exhaustivos deben sumar la unidad:

$$\sum_{i=1}^n \text{Prob}(x_i) = 1$$

  • Probabilidad conjunta: Los eventos ocurren simultaneamente.
$$\text{Prob}(F1~\text{and}~V2)$$

  • Probabilidad condicional: probabilidad de que ocurra un evento sabiendo que el otro ya ocurrio.
$$\text{Prob}(V2 ~ | ~F1) = \text{Prob}(F1~\text{and} ~ V2) ~/~\text{Prob}(F1)$$

  • Independencia: Si los eventos $A$ y $B$ son independientes:
$$\text{Prob}(A~|~B) = \text{Prob}(A)$$

De la definicin de probabilidad condicional:

$$\text{Prob}(A~|~B) = \text{Prob}(A) = \frac{\text{Prob}(A~\text{and}~B)}{\text{Prob}(B)}$$

Entonces:

$$\text{Prob}(A~\text{and}~B)~=~\text{Prob}(A)~*~\text{Prob}(B)$$

  • Probabilidad marginal: Sea $_i$ con $ = 1, ... , $, un conjunto de eventos mutuamente exclusivos y colectivamente exhaustivos. La probabilidad de un evento $$ es:
$$\text{Prob}(A) = \sum_{i=1}^n \text{Prob}(A~\text{and}~X_i)$$

  • Para dos eventos $A$ y $B$:
$$\text{Prob}(A ~\text{or}~B) = \text{Prob}(A) + \text{Prob}(B) - \text{Prob}(A~\text{and}~B)$$

  • Complemento: $$\text{Prob}(\text{not}~ A) = 1 - \text{Prob}(A)$$
  • Probabilidad total: $$\text{Prob}(A) = \text{Prob}(A~\text{and}~B) + \text{Prob}(A~\text{and}~\text{not}~B) = \text{Prob}(A~|~B)~\text{Prob}(B) ~+~ \text{Prob}(A~|~\text{not}~B)~\text{Prob}(\text{not}~B)$$
  • Teorema de Bayes: $$\text{Prob}(A~\text{and}~B) = \text{Prob}(A~|~B)~\text{Prob}{B} = \text{Prob}(B~|~A)~\text{Prob}{A}$$
$$\text{Prob}(B~|~A) = \frac{\text{Prob}(A~|~B)~\text{Prob}{B}}{\text{Prob}(A)}$$$$ = \frac{\text{Prob}(A~|~B)~\text{Prob}{B}} {\text{Prob}(A~|~B)~\text{Prob}(B) ~+~ \text{Prob}(A~|~\text{not}~B)~\text{Prob}(\text{not}~B)}$$

Representación de la probabilidad para variables continuas¶

Se usa una funcin matemtica para definir la funcin de densidad de probabilidad. Se nota como $f_X(x)$.

La funcin de densidad de probabilidad acumulada es la integral de $f_X(x)$ entre $-\infty$ y $x$ y se define como:

$$F_X(x)= \int_{-\infty}^x f_X(u) du$$

Momentos de una distribución¶

Los momentos de una distribucin permiten caracterizar sus propiedades. Estos son:

Media o valor esperado: $ E[x_t ]= _x =$ $ \int f_x (u)\,du =M_1$

Varianza: $V[x_t ] = _x^2 = E[(x_t-_x )^2 ]=M_2-M_1^2$

Asimetra: $S_x = E[(x_t-_x )^3 ]=M_3-3M_2 M_1+2M_1^2$

Curtosis: $K_x = E[(x_t-_x )^4 ]=M_4-4M_3 M_1+6M_2 M_1^2-3M_1^4$

Principales distribuciones continuas¶

  • Distribucin normal:
$$f_X(x|\mu_X, \sigma_X) = \frac{1}{\sigma_X \sqrt{2\pi}} \exp\Big[ {-\frac{1}{2}\Big( \frac{x-\mu_X}{\sigma_X} \Big)^2} \Big]$$
  • Distribucin normal estandarizada: es la distribucin normal con $\mu_X=0$ y $\sigma_X=1$.
  • Distribucin lognormal: La variable $y = \log(x)$ sigue una distribucin normal.

Propiedades útiles de la distribución normal¶

  • Sean $x \sim N(\mu_X, \sigma_X^2)$ y $y \sim N(\mu_Y, \sigma_Y^2)$, entonces:
$$ x + y \sim N(~\mu_X~+~\mu_Y, ~\sigma_X^2 + \sigma_Y^2~)$$
  • Sea $x \sim N(\mu_X, \sigma_X^2)$ y dos nmeros reales $a$ y $b$, entonces:
$$a~+~b~\sim N(a\mu_X+b,~ a^2 \sigma_X^2)$$
  • Sean $x \sim N(\mu_X, \sigma_X^2)$ y $\epsilon \sim N(0, 1)$, entonces:
$$x = \mu_X~+~\sigma_X \epsilon$$

Muestreo de una distribución de probabilidades¶

La muestra ${x_1, x_2, ..., x_n}$ se define como una muestra aleatoria de una distribucin de probabilidades, si los valores $x_i$:

  • Tienen la misma distribucin de probabilidades.
  • Son independientes.

Estas propiedades se suelen abreviar como iid (independientes e identicamente distribuidas).

Pruebas de normalidad¶

Un problema tpico de series de tiempo consiste en determinar si una muestra de datos proviene de una distribucin normal. Para ello, se consideran las siguientes hiptesis:

  • $H_0:$ Las observaciones ${x_1,,x_T}$ provienen de una distribucin normal con media y varianza desconocidas
  • $H_a:$ Las observaciones provienen de una distribucin arbitraria

Metodologas tpicas:

  • Grfico Cuartil Cuartil (Q-Q plot)
  • Grfico de probabilidad normal
  • Prueba de Jarque-Bera

Grfica Cuartil-Cuartil (Q-Q plot)

Mtodo grfico para comparar dos distribuciones. Si los datos provienen de la misma distribucin, forman una lnea recta.

Prueba de normalidad de Jarque-Bera

Se basa en determinar si los datos tienen la misma asimetra $(M_3)$ y curtosis $(M_4)$ de una distribucin normal.

  • $H_0:$ Los datos provienen de una distribucin normal
  • $H_a:$ Los datos no provienen de una distribucin normal

Esta prueba puede ser engaosa para muestras pequeas. El rechazo de $H_0$ puede indicar:

  • La presencia de observaciones atpicas
  • Que la varianza no es constante
  • La presencia de comportamiento no lineal (dependencia temporal)

Teorema del Límite Central¶

Sea ${x_i}$ para ${i=1,,n}$ una muestra aleatoria $i.i.d.$ con media $$ y varianza $^2$. Para $n\infty$ la variable ${S_n=}$ $\frac{1}{n}$ $\sum_{i=1}^{n}x_{i} $, se aproxima a una distribucin normal con media $$ y varianza $\frac{^2}{n}$

Bootstrap¶

El bootstrap es una tcnica numrica para estimar la distribucin de probabilidades de cualquier estadstico calculado sobre una muestra de datos y tiene muchas aplicaciones en modelado predictivo y pronstico de series de tiempo.

Sea una secuencia $\{x_i; i = 1, ...,n\}$ proveniente de una distribucin poblacional $F$ desconocida con media $\mu$ y varianza $\sigma^2$. Esto es:

$$ F \to (x_1, ...,x_n)$$

  • $F$ es aproximada a travs de la distribucin emprica de los datos $G$.

  • El parmetro $\theta$ es una funcin $()$ de la distribucin de probabilidades de la poblacin $: = ()$

  • $\theta$ se aproxima a partir de la distribucin emprica de probabilidad de los datos: $\theta = ()$.

  • Muestra bootstrap: Sea G la distribucin emprica de los datos que coloca una probabilidad de $1/n$ en el dato $_i, ~=1,...,$.Una muestra bootstrap se define como una muestra aleatoria de tamao $$ obtenida de $G$:

$$^ = ( _1^, _2^, ..., _n^*)$$

por ejemplo, $_1^ = _3, _2^= _1, _3^ = _3, ...$

Algoritmo bootstrap no paramtrico

  • Se tiene la muestra $x=\{x_i; 1,...,n\}$
  • Se seleccionan $B$ muestras bootstrap independientes $x^{*1}, ..., x^{*B}$ cada una de $n$ elementos obtenidos de la muestra original.
  • Para cada muestra bootstap se calcula el estadstico de interes: $\theta(b) = t(x^{*b})$
  • La muestra $\theta(1), ..., \theta(B)$ representa la distribucin de probabilidades de $\theta$ y puede calcularsele valor esperado, desviacin estndar, etc.
